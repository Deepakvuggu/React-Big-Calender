# React Big Calendar + Bar Graph + Redux


## Features
- Fully integrated **Redux Toolkit** for managing global calendar state.
- Big Calendar with highlighted dates that contain data.
- Modal popup on date selection.
- Bar graph shows user-based values for selected date.
- “No data found” warning for empty days.


## Tech Stack
- React
- Redux Toolkit
- React Big Calendar
- date-fns
- Recharts
- React Modal


## Setup
1. `npm install`
2. `npm start`


Redux store: `/src/store/`
